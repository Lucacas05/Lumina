# Referencias visuales locales

Los mockups, referencias HTML y materiales de exploracion visual no deben vivir dentro de `src/pages`.

## Ubicacion

- `references/`: referencias no publicadas por Astro

## Regla

Si una referencia se quiere conservar para inspirar una pantalla futura, se mueve a `references/` o a otra carpeta fuera del enrutado de Astro.
